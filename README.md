# L’usage des réseaux sociaux influence-t-il les résultats scolaires ?

Application web réalisée avec Oracle APEX dans le cadre d’un projet de cours, en binôme.

## Description

Ce projet a pour objectif d’étudier s’il existe un lien entre l’usage des réseaux sociaux et les résultats scolaires.  
L’application s’appuie sur un jeu de données anonymisé contenant les réponses d’étudiants de plusieurs pays et de différents niveaux d’études.  
Les données décrivent leurs habitudes d’utilisation des réseaux sociaux, leurs préférences de plateformes et certaines conséquences dans leur vie scolaire et personnelle.

## Objectif

L’objectif est d’analyser :
- l’intensité d’utilisation des réseaux sociaux ;
- les comportements liés à l’addiction ;
- l’impact possible sur les résultats scolaires ;
- les différences selon le niveau d’étude ou le profil des étudiants.

## Jeu de données

Le projet utilise le corpus **Student’s Social Media Addiction** disponible sur Kaggle.

Lien du dataset :  
https://www.kaggle.com/datasets/adilshamim8/social-media-addiction-vs-relationships/data

## Fonctionnalités

- Visualisation des données.
- Analyse des habitudes d’usage des réseaux sociaux.
- Étude du lien entre usage numérique et résultats scolaires.
- Interface web développée avec Oracle APEX.
